# Claude Code Starter Guide

Welcome! This guide will help you go from zero to productive with Claude Code - even if you've never used a terminal or written code before.

## What is Claude Code?

Claude Code is an AI-powered command-line tool that helps you build software by having a conversation. You describe what you want to build, and Claude Code helps you create it, step by step.

## What You'll Learn

This guide will walk you through:

1. **Installing Claude Code** - Get the tool set up on your computer
2. **Setting up GitHub** - Create an account to store and share your code
3. **Using the Terminal** - Learn basic commands to navigate and work with files
4. **Your First Project** - Build something real with Claude Code
5. **Pushing to GitHub** - Save your work online

## Who Is This For?

This guide is designed for **non-programmers** who want to:
- Build tools to help with their job
- Automate repetitive tasks
- Explore money-making opportunities using AI
- Learn by doing

No prior coding experience required!

If you're comfortable running scripts, we provide automated setup:

```bash

```

**Windows (PowerShell):**
```powershell
.\scripts\setup-windows.ps1
```
